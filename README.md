# 23ビット加算器表示器

22個の全加算器と1個の半加算器を用いて、最大 (2^23 - 1) + (2^23 - 1) の加算結果を7セグメントLEDで表示する電子回路です。

## 概要

半加算器・全加算器をブレッドボード上に組み上げ、2つの入力値の加算結果を8桁の7セグメントLEDに表示します。Arduino Nano Everyでダイナミック点灯制御を行っています。

## 背景

大学のエクステンションセンターで自ら企画した小学生向け電子工作講座の教材として制作しました。「コンピュータがどうやって足し算をするのか」を実物で体験してもらうことを目的としています。半加算器・全加算器を自分で作り、それを繋げていくと大きな桁の計算ができるという加算器の原理を、実際に手で触れながら学べる教材を目指しました。

## 使用マイコン

Arduino Nano Every

## 技術スタック

| 技術 | 用途 |
|------|------|
| C++ / Arduino | マイコン制御・ダイナミック点灯 |
| トランジスタ(2sc1815) | スイッチング回路 |
| シフトレジスタ(TC74HC165AP) | 出力ピン数の拡張 |
| 7セグメントLED | 数値表示 |

## 回路の構成

- 半加算器 × 1、全加算器 × 22 をブレッドボード上に実装
- シフトレジスタを用いてArduinoの出力ピン数を拡張
- ダイナミック点灯とトランジスタのスイッチングにより8桁の7セグメントLEDを制御

## 学び

トランジスタによるスイッチング、ダイナミック点灯、シフトレジスタを用いた出力拡張など、高度な電子回路設計技術を実装を通じて習得しました。論理回路の理論が実際の電子部品として動作する体験は、大学の講義で学んだアナログ回路の知識と直結するものでした。
